# 硕士论文写作一页检查表（计算机类）

说明：此检查表为一页速查版，覆盖从题目到提交的关键项。可打印并夹在草稿或提交材料中。

## 基本信息
- [ ] 题目（中文/英文，简洁准确）
- [ ] 作者、导师、学院、提交日期、学位类型已正确填写并符合模板
- [ ] 摘要（中文）包含：问题 + 方法 + 主要结果（量化） + 贡献（简短）
- [ ] Abstract（英文）已翻译并校对
- [ ] 关键词（3–6 个）

## 引言（Introduction）
- [ ] 背景与动机：说明问题重要性
- [ ] 问题陈述：清楚、可操作
- [ ] 现有方法不足：简短指明 gap
- [ ] 贡献清单（3–5 条，量化指标）
- [ ] 论文结构一句话概述

## 相关工作（Related Work）
- [ ] 按主题/方法分类回顾（非逐篇罗列）
- [ ] 关键工作对比表或图（方法 / 数据集 / 性能 / 局限）
- [ ] 明确指出研究差距

## 背景/预备知识（Background）
- [ ] 提供必要定义、记号、评价指标
- [ ] 简述关键理论或工具（只写必要部分）

## 方法（Methodology）
- [ ] 问题形式化（输入/输出/目标）
- [ ] 方法框架图或流程图
- [ ] 关键算法伪代码或步骤说明
- [ ] 参数与超参数说明（默认值）
- [ ] 若有理论结果，正文结论 + 证明放附录

## 实验设置（Implementation / Experiments）
- [ ] 数据集来源与划分（train/val/test）与统计量表
- [ ] 评价指标与基线方法列明
- [ ] 硬件与软件环境（GPU 型号、框架版本）
- [ ] 超参数表与调参策略
- [ ] 随机种子、运行次数（或交叉验证方式）

## 结果与分析（Results）
- [ ] 与基线的量化对比表（含单位/置信区间/显著性）
- [ ] 训练/验证曲线或收敛图
- [ ] 消融实验表明各模块贡献
- [ ] 典型成功/失败案例与分析

## 讨论与结论（Discussion & Conclusion）
- [ ] 总结主要发现（不要引入新结果）
- [ ] 方法的局限性与适用范围
- [ ] 清晰列出 2–4 条可行的未来工作

## 参考文献与附录
- [ ] 引用格式统一且完整（DOI/URL/访问日期）
- [ ] 附录包含：证明、额外图表、代码运行说明或数据处理脚本

## 可复现性与提交材料
- [ ] 代码仓库链接已准备（README 包含运行步骤）
- [ ] requirements.txt / environment.yml 列出依赖版本
- [ ] 训练与评估示例命令（可复制）
- [ ] 随机种子、实验时长、显存需求说明

## 格式与质量核查
- [ ] 所有图表有编号、标题、来源说明和讨论
- [ ] 表格小数点/单位规范，列名清晰
- [ ] 语法拼写检查完成（建议两轮校对）
- [ ] 导师/同学已审阅并提出修改意见
