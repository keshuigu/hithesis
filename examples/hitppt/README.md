# LaTeX博士答辩PPT编译工具

## 编译环境要求

- XeLaTeX编译器
- 中文字体支持（SimSun等）
- Beamer包和相关依赖包

## 编译命令

```bash
# 编译PPT
make

# 清理临时文件
make clean

# 完全清理（包括PDF）
make distclean
```

## 文件结构

- `defense_presentation.tex` - 主PPT文件
- `figures/` - 图片资源目录
- `Makefile` - 编译配置文件

## PPT内容结构

1. **标题页** - 论文标题和作者信息
2. **目录** - 答辩内容概览
3. **研究背景与意义** - 问题背景和研究动机
4. **研究现状及存在问题** - 技术现状和挑战分析
5. **研究内容与贡献** - 主要创新点和贡献
6. **相关理论与技术基础** - 理论基础介绍
7. **模板逆向攻击方法(TIA)** - 第一个核心方法
8. **模型反演攻击方法(MIA)** - 第二个核心方法  
9. **实验结果与分析** - 实验验证和性能分析
10. **结论与展望** - 总结和未来工作

## 自定义修改

### 修改个人信息
编辑`defense_presentation.tex`文件中的以下部分：
```latex
\title[短标题]{完整论文标题}
\author[答辩人]{答辩人姓名}
\institute[HIT]{单位信息}
\date{\today}
```

### 添加图片
将图片文件放入`figures/`目录，然后在PPT中使用：
```latex
\includegraphics[width=0.5\textwidth]{figures/your_image.png}
```

### 调整内容
根据实际论文内容修改各章节的具体内容，特别注意：
- 研究背景和意义
- 实验数据和结果
- 创新点和贡献
- 结论和展望

## 注意事项

1. 确保安装了中文字体，如SimSun
2. 使用XeLaTeX编译器以支持中文
3. 图片建议使用PNG或JPG格式
4. PPT总时长控制在20-30分钟内
5. 重点突出创新点和贡献

## 编译问题解决

如果编译出现问题：
1. 检查XeLaTeX是否正确安装
2. 确认中文字体是否可用
3. 检查相关LaTeX包是否安装
4. 查看编译日志文件定位具体错误
